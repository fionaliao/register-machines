Encoder

Input a program.

Example program:
R0- > L3, L1;
HALT;
R0+ > L0;
R0- > L0, L1

Note: Encoder does not support explicit labelling, so "L0: R0- > L3, L1" is not valid.
Also, semicolons separate each label line and are not used after the last label line unless the user is wishing to add an
implicit HALT statement after the last label line.

Decoder

Input a number. Expressions in the format x * y and x ^ y are also allowed. Evaluation is power first then multiplication. No brackets.
